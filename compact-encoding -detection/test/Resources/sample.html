Shift JISの文字コードを指定する

説明

ページ内で使用される文字コードはhttp-equivにcontent-typeを指定します。contentにcharset=text/html;shift_jisを指定します。古い時代に作成されたHTML文書の中にはShift JISコードをx-sjisと記述している場合があります。

サンプルコード [実行]

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN">
<html>
<head>
<meta http-equiv="content-type" content="text/html;charset=shift_jis">
<title>Sample</title>
</head>
<body>
<h1>Sample</h1>
<p>
文字コードを指定します。
</p>
</body>
</html>
